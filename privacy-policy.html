Privacy Policy for My n8n Automation Tool

This document outlines the privacy policy for the application "My n8n Automation Tool".

1. Data Collection and Usage:

This application is an internal, private tool designed solely for the purpose of automating video uploads to the owner's personal TikTok account. We do not collect, store, share, or process any personal data from users of TikTok. The application only handles the authentication token of the app owner to facilitate the API connection with TikTok, as per TikTok's official documentation.

2. Data Storage:

The authentication token is stored locally on the owner's machine and is not accessible to any third parties.
